
一、数据库（DATABASE）
--------------------------------------------------
增：CREATE DATABASE db_name CHARACTER SET utf8mb4;
删：DROP DATABASE IF EXISTS db_name;
改：ALTER DATABASE db_name CHARACTER SET utf8mb4;
查：SHOW DATABASES;  SELECT DATABASE();

二、数据表（TABLE）
--------------------------------------------------
增：CREATE TABLE t (id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(50));
删：DROP TABLE IF EXISTS t;
改：ALTER TABLE t ADD COLUMN age INT, MODIFY name VARCHAR(100), RENAME TO new_t;
查：SHOW TABLES;  DESC t;  SHOW CREATE TABLE t;

三、约束（CONSTRAINT）
--------------------------------------------------
增：ALTER TABLE t ADD CONSTRAINT pk_t PRIMARY KEY(id);
     ALTER TABLE t ADD CONSTRAINT fk_t FOREIGN KEY(uid) REFERENCES u(id);
删：ALTER TABLE t DROP PRIMARY KEY;  ALTER TABLE t DROP FOREIGN KEY fk_t;
改：先删后建（无直接修改语法）
查：SELECT * FROM information_schema.TABLE_CONSTRAINTS WHERE table_schema='db';

四、数据（DML/DQL）
--------------------------------------------------
增：INSERT INTO t(name,age) VALUES ('Tom',18),('Jerry',19);
删：DELETE FROM t WHERE id=1;   TRUNCATE TABLE t;  -- 整表快速清
改：UPDATE t SET age=20 WHERE name='Tom';
查：SELECT * FROM t WHERE age>=18 ORDER BY id DESC LIMIT 10;

快捷记忆：
库→CREATE/DROP/ALTER/SHOW
表→CREATE/DROP/ALTER/DESC
约束→ADD/DROP + information_schema
数据→INSERT/DELETE/UPDATE/SELECT
