| 方案      | 优点            | 缺点            | 适用场景          |
| ------- | ------------- | ------------- | ------------- |
| **状态栈** | 自动管理历史，返回逻辑简单 | 内存占用略高（保存上下文） | **层级菜单、向导流程** |
| 简单变量    | 实现简单          | 无法记忆历史，返回困难   | 只有2-3层的简单菜单   |
| 状态机     | 结构清晰          | 返回逻辑需手动实现     | 线性流程（无频繁返回）   |
| 树形导航    | 可视化好          | 返回需回溯树路径      | 静态菜单结构        |
