安装jdk:
      1,用root用户登录
	  2，卸载centos7自带的jdk,centos7以前的版本没有自带jdk
	  3,查看自带jdk安装路径   rpm -qa | grep java
	  4,删除自带jidk    
		rpm  -e  --nodeps    java-1.8.0-openjdk-1.8.0.262.b10-1.el7.x86_64
		rpm  -e  --nodeps    java-1.8.0-openjdk-headless-1.8.0.262.b10-1.el7.x86_64
	  5,查看是否卸载成功(输入如下命令如果出现
	  -bash: /usr/bin/java: 没有那个文件或目录就表示成功)
	  
	     java --version
		 
	  6，进入/usr目录    cd /usr
	  7，创建文件夹    mkdir   java
	  8,进入java文件夹  通过ssh工具把jdk安装包传入
	  9，解压：tar -zxvf jdk-8u162-linux-x64.tar.gz -C /usr/java
	
	   10,配置环境变量
	      打开  etc/profile这个文件： vi   /etc/profile
		  export JAVA_HOME=/usr/java/jdk1.8.0_162
		  export JRE_HOME=${JAVA_HOME}/jre
		  export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
		  export  PATH=${JAVA_HOME}/bin:$PATH

		12，配置的环境变量立即生效：
		     source   /etc/profile

安装mysql:
	Could not retrieve mirrorlist http://mirrorlist.centos.org/?release=7&arch=x86_64&repo=os&infra=stoc
	CentOS 7
　　　　    wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
	前言：先安装wget命令
		yum -y install wget
	          查看之前安装过MySQL没有
		查询已安装的mysql rpm -qa |grep mysql
	          如果之前安装过，要把之前的卸载掉
		rpm -e --nodeps mysql 强力删除模式，用该命令强力删除
	
	1、wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm（下载rpm）
	2、rpm -ivh mysql-community-release-el7-5.noarch.rpm（安装rpm包）
	3、yum list | grep mysql 来查看安装的mysql 的rpm包
	4、yum install -y mysql-server mysql mysql-devel进行安装
	5、启动mysql服务service mysqld start   systemctl start mysqld
	6、将mysqld服务加入到开机自启动项中。命令：chkconfig mysqld on
	7、修改mysql 的密码：mysqladmin -u root password 'root'
	8、数据库登录。命令：mysql -u root -p 登录数据库

	9、授权
		--赋予任何主机访问权限：
			GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'password' WITH GRANT OPTION;
		--允许指定主机(IP地址)访问权限：
			GRANT ALL PRIVILEGES ON *.* TO 'myuser'@'192.168.1.3' IDENTIFIED BY 'root' WITH GRANT OPTION;
		通过GRANT命令赋权后,需要通过FLUSH PRIVILEGES刷新权限表使修改生效：
		flush privileges;

		注：为了使远程客户端能访问成功，需将端口3306加入防火墙白名单：
		先退出mysql,     输入exit   退出mysql
		firewall-cmd --add-port=3306/tcp --zone=public --permanent

		重启防火墙：firewall-cmd --reload
安装tomcat:
           1,进入/usr目录
		   cd /usr
		   2，创建tomcat文件
		   mkdir tomcat
		   3,进入tomcat文件夹
		     cd tomcat
		   4,上传tomcat安装包到这个文件夹
		   5，解压tomcat
		   tar -zxvf  tomcat安装包名
		   6，给解压后的文件改个名
		      mv   旧名   新名
		   7,进入tomcat的bin目录
		       cd   新名
			   cd   bin
			8,启动tomcat
			    ./startup.sh
			
			    
			  8.3在本机打开浏览器输入服务器的ip加上8080，访问出现未建立连接
			  是因为防火墙拦截了这个端口号，关闭防火墙即可，步骤如下：
			   8.3.1：停止tomcat
			        ./shutdown.sh或sh  shutdown.sh
			   8.3.2:关闭防火墙
			       systemctl stop  firewalld
			   8.3.3:开启tomcat
			     ./startup.sh或sh startup.sh
				8.3.4:在本机浏览器输入ip地址加端口号即可访问出现tomcat页面表示成功
				     注意：此处的ip是虚拟机ip
