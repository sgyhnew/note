1.sql语言
	DDL数据库定义语言：查询数据库和表对象
		CREATE/DROP/ALTER/TRUNCATE
	DML数据操作语言：
		INSERT/UPDATE/DELET
		insert into 表名[(列名)] values(值1,值2…,值值n)
		delete from表名 where 条件
		update 表名 set 列名 = 值,列名=值…,列名=值 where 条件
		insert into tb,update tb set
	DQL数据定义语言：
		SELECT
	DCL数据控制语言：权限管理
		GRANT/REVOKE
	TCL事务控制语言：
		COMMIT、ROLLBACK、SAVEPOINT
2.sql约束
	主键约束
		主键名，数据类型，AUTO_INCREMENT PRIMARY KEY,     自增+主键
		不可为空，不能重复
	外键约束
			alter table 从表 add constraint 约束名(任意) foreign key(外键名) references 主表(主键名)
		保持数据一致性和完整性
		外键是一个主表的从表，一个主表可有多个外键，外键可为空且数据类型需与主键一致
	其他
		列名 数据类型 + 
			默认约束 default 'str' 
			唯一约束 unique 不能重复可为空
			非空约束 not null
			检查约束 check (列名 判别式)
			auto_increment 自增长
			comment 'str' 注释
3.空与真空
	空，即空字符串，在硬盘中预留空间但值为空值
	真空，即广义上的‘空’，不在硬盘中预留空间

	alter table tb drop 列/约束
