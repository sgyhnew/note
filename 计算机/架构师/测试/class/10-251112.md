1.多表联查
	select 表1.字段1，表2.字段2 from 表1，表2 where 表1.字段=表2.字段，条件
	select 表1.字段1，表2.字段2 from 表1 inner join 表2 on 表1.字段=表2.字段 where 条件
2.表的连接
	1.内连接inner join
	2.左连接left join
		：左表完全展示，右表满足条件展示，不满足则以null补空
	3.又连接right join
	4.全连接union
		：多个连接语句中加入union，把连接的查询结果去重且并和。等价于不存在的`full join`	
3.子查询
	select 字段1 from 表1，(select 字段2 from 表2 where 条件) 临时表 where 条件
	在查询中嵌套一个查询，称该查询子查询，其结果为被视作一个临时表。必须在括号内且为临时表命名
