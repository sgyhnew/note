| 类型     | 可变性       | 示例                    |
| :----- | :-------- | :-------------------- |
| **列表** | ✅ **可变**  | `lst[0] = 10`         |
| **元组** | ❌ **不可变** | `tup[0] = 10` → 报错    |
| **字典** | ✅ **可变**  | `dict["key"] = value` |
| **集合** | ✅ **可变**  | `set.add(4)`          |
| **数组** | ✅ **可变**  | `arr[0] = 10`         |

| 容器类型           | **是否允许重复元素**              | **重复时行为** | **示例**                     |
| :------------- | :------------------------ | :-------- | :------------------------- |
| **列表 (List)**  | ✅ **允许**                  | 保留所有重复项   | `[1, 1, 2] → [1, 1, 2]`    |
| **元组 (Tuple)** | ✅ **允许**                  | 保留所有重复项   | `(1, 1, 2) → (1, 1, 2)`    |
| **字典 (Dict)**  | **键不允许重复**  <br>**值允许重复** | 键重复会覆盖旧值  | `{"a":1, "a":2} → {"a":2}` |
| **集合 (Set)**   | ❌ **不允许**                 | **自动去重**  | `{1, 1, 2} → {1, 2}`       |
| **数组 (Array)** | ✅ **允许**                  | 保留所有重复项   | `array('i', [1,1,2])`      |

| 类型     | Python 3.7+ | 索引访问                |
| :----- | :---------- | :------------------ |
| **列表** | ✅ 有序        | ✅ `lst[0]`          |
| **元组** | ✅ 有序        | ✅ `tup[0]`          |
| **字典** | ✅ 有序（插入顺序）  | ✅ `dict["key"]`（按键） |
| **集合** | ❌ 无序        | ❌ `set[0]` → 报错     |
| **数组** | ✅ 有序        | ✅ `arr[0]`          |

|特性|列表|元组|字典|集合|数组|
|:--|:--|:--|:--|:--|:--|
|**符号**|`[]`|`()`|`{}`|`{}`|`array()`|
|**空对象**|`[]`|`()`|`{}`|`set()`|`array('i')`|
|**元素类型**|任意|任意|键:任意不可变  <br>值:任意|任意不可变|**统一数值类型**|
|**访问方式**|索引 `lst[i]`|索引 `tup[i]`|按键 `dict[key]`|不可索引|索引 `arr[i]`|
|**查找速度**|O(n)|O(n)|**O(1)**（哈希）|**O(1)**（哈希）|O(n)|
|**内存效率**|一般|较高|较低|较低|**最高**（纯数值）|
|**典型场景**|通用序列|固定数据|映射关系|去重/集合运算|大数据量数值计算|
