2.linux基础命令
	1.vi/vim 文件 文件编辑器
		i  在当前位置编辑
		o 在当前光标下一行编辑
		a 在当前位置后面编辑
		: 输入命令
		set nu 显示文件行数
		G  将光标移动到文件最后一行
		gg 将光标移动到文件第一行
		dd 删除改行数据
	2.tar 压缩解压命令
		-c 压缩
		-x 解压
		-t 查看
		-f 指定归档文件的文件名
		-v 执行时显示详细信息
		-z 使用gzip压缩
		-C 指定解压到的目录
		-k 保持不文件，不覆盖已存
		实例: tar -zxvf a.tar.gz -C /path 将文件a.tar.gz解压到path目录
		zip/unzip无需参数
	3.cat/more/less/head/tail/wc查看文件
		cat
			cat 文件                    输出内容到终端
			cat 文件1 文件2        依此输出文件1和文件2
			cat 文件1 > 文件2     文件1覆盖文件2
			cat 文件1 >> 文件2  文件1追加文件2末尾    
		more
			空格 向下翻页
			enter 向下滚动一行
			/a 向下搜孙a字串
			q 推出
		less
			+F 向下翻页
			+B 向上翻页
			+D 向下翻半页
			+U 向上
			翻半页
			/a 向下搜寻a字串
			?a 向上搜寻a字串
			q 推出
		head 
			+文件   显示文件前10行
			-n num 文件   显示文件前num行
			-c num 文件   显示文件前num字节
			-q 文件1 文件2 显示多个文件开头但不显示文件名
			-v 文件1 文件2 显示多个文件开头且显示文件名
		tail
			+文件名 显示文件最后10行
			-n num 文件   显示文件前num行
			-c num 文件   显示文件前num字节
			-q 文件1 文件2 显示多个文件开头但不显示文件名
			-v 文件1 文件2 显示多个文件开头且显示文件名
			-f 文件 持续显示文件新增内容，常用于查看日志文件的实时更新
		wc
			-文件名 统计文件名中的行数、字符数、字节数
			-l 统计行数
			-w 统计单词数
			-c 统计字节数
			-m统计字符数(包含空白总字符)
			-L 显示最长行的长度
	4.grep
		grep ‘a’ 文件  搜索文件中含字符a的行
		-i 忽视大小写
		-v 反向匹配
		-n 显示行号
		-c统计匹配行数
		- l 只输出匹配的文件名
		- r/R 递归
		- w 整词匹配
		- E 扩展正则表达式
	5.history 查看历史命令
	6.find
		-name file.txt 在当前路径查找file文件
		/ -name file.txt 全盘查找
		. -name file.txt 在当前路径以及其字路径
		-type 类型
			f 文件，d 文件夹，l链接
		-size 大小
			+n 大于n个单位，-n小于n个单位，n等于n个单位
			c字节、k千字节、M兆字节、G
		-perm 权限
			读r4、写w2、执行x1，ugo
3.linux进程命令
	1.ps 显示当前进程
		PID：进程ID  
		TTY：终端设备  
		TIME：进程运行时间 START:  进程启动时间  
		CMD：进程命令行 
		UID：进程所有者ID  PPID：父进程ID  
		C：cpu使用率          %CPU：CPU 使用率         %MEM：内存使用率 
		VSZ：虚拟内存大小  RSS：物理内存大小（以 KB 为单位）
		STAT  :进程状态(s代表体眠，R表示运行，z表示僵尸状态)
		-e 显示系统中所有进程
		-a 显示当前终端的进程
		-f 显示长格式详细信息
		-x 显示所有没有控制终端的进程
		-u 用户名 显示特定用户进程，默认为当前用户
		-g 组名 显示特定组
		-p 父进程id --forest 显示特定父进程的子进程
		例：ps -aux 、ps -ef。如果需要查看进程占用的 cpu,内存的使川情况,使用 ps-aux。如果只查看进程信息,使用 ps-ef
	2.top 实时查看进程信息
	3.kill 杀死进程
		-9 pid 强制杀死进程
	4.df -h 查看磁盘整体使用情况
	   du-h 当前目录文件占有磁盘空间
	5.netstat 查看端口
		-a 显示所有活动的网络连接
		-l 显示监听的端口
		-n 显示网络连接的详细信息
		-p 显示每个连接的进程ID和程序名
		-tulnp 显示所有监听以及其对应的程序名
		-ano 显示所有活动的网络连接，以数字形式显示地址和端口并显示连接状态
		-t/u  tcp/udp端口
