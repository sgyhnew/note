1.为mysql赋予被其他用户访问的权限
	1.修改 root 用户的密码 ./mysqladmin -u root password 'root'
	2.登录 root 用户./mysgl -u root -p
	3.为root 用户授权远程访问权限GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'password' WITH GRANT OPTION;
	4.刷新权限flush privileges;
